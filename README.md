This code is written for the Wireless Wanderlust hackathon at ISIT 2025 by the Signal Sifters team (Spencer Hill and Ilir Gusija). 

It implements a semi-supervised learning algorithm, called channel charting, to determine the location of a receiver based on its channel-state information
with an array of wireless receivers. 

The training files are in src/scripts, which contains train.py (which trains the first version of our model) and train_channel_chart.py, which is the semi-supervised version.
